# Antigravity 懶人包 #04.5：連接 Firebase 資料庫 (Windows PowerShell 版)

本懶人包已針對 **Antigravity** 進行優化與轉化。Antigravity 可以自動辨識、讀取此檔案，並在您的授權下自動在 PowerShell 終端機中執行相關安裝、規則部署與 MCP 設定指令。

---

## 這個懶人包會幫你做什麼？

讓 Antigravity 能夠直接操控 Firebase Cloud Firestore：
- 自然語言建立集合（Collections）與進行資料 CRUD（增刪查改）
- 即時數據同步（Real-time Updates）
- 教學工具、千人研習或即時課堂互動網頁（如即時文字雲、課堂投票）的數據連接
- 支援免費方案（Spark 方案：專案數無限、不使用不閒置暫停、並發連線高）

---

## 先備條件

- [x] **Node.js 與 npm**: 已安裝
- [x] **Google 帳號**: 用於登入 Firebase
- [x] **網路連線**: 保持暢通

## 執行步驟說明

### 步驟零：環境與登入檢查
1. 確保 Node.js 正常運作 (`node --version`)。
2. 檢查並登入 Firebase CLI。若尚未登入，請於 PowerShell 中執行：
   ```powershell
   npx.cmd firebase-tools login
   ```
   *(會開啟瀏覽器進行 Google 帳號授權登入)*
3. 驗證登入成功與專案列表：
   ```powershell
   npx.cmd firebase-tools projects:list
   ```

### 步驟一：在 Firebase Console 建立專案
1. 瀏覽至 [Firebase Console](https://console.firebase.google.com/) 建立新專案（例如：`my-teaching-tools`）。
2. 在左側選單點擊 **Firestore Database**，然後點擊 **建立資料庫**。
3. 選擇以 **正式版模式** 啟動（安全性規則預設為禁止所有讀寫）。
4. 選擇位置（例如 `asia-east1` 台灣 或 `asia-northeast1` 東京）。

### 步驟二：配置本機專案設定檔
在專案根目錄建立以下三個核心檔案以利後續部署規則與資料同步：

1. **`firestore.rules`** (安全規則配置，放行特定集合如 `wordcloud_words`):
   ```javascript
   rules_version = '2';
   service cloud.firestore {
     match /databases/{database}/documents {
       match /wordcloud_words/{document} {
         allow read, write: if true;
       }
       match /{document=**} {
         allow read, write: if false;
       }
     }
   }
   ```
2. **`firebase.json`** (指定規則檔案):
   ```json
   {
     "firestore": {
       "rules": "firestore.rules"
     }
   }
   ```
3. **`.firebaserc`** (指定預設專案 ID):
   ```json
   {
     "projects": {
       "default": "你的 Firebase 專案 ID"
     }
   }
   ```

### 步驟三：部署 Firestore 安全性規則
執行以下指令將規則部署到 Firebase 雲端：
```powershell
npx.cmd firebase-tools deploy --only firestore:rules
```
部署成功後會看到 `Deploy complete!`，您的專案即可開始對指定的 Collections 進行安全讀寫。

### 步驟四：將 Firebase 註冊至 Antigravity MCP 伺服器
編輯全域 MCP 設定檔 `C:\Users\User\.gemini\antigravity\mcp_config.json`，在 `mcpServers` 區段下新增 Firebase 伺服器設定：

```json
{
  "mcpServers": {
    "firebase": {
      "command": "npx.cmd",
      "args": ["-y", "firebase-tools@latest", "mcp"]
    }
  }
}
```
**配置完成後，請完整重啟 Antigravity 對話環境，使 Firebase 工具生效。**

## 常見問題與異常排除

| 狀況 | 原因 | 解法 |
| :--- | :--- | :--- |
| `npx: 找不到指令` | 未安裝 Node.js | 請安裝 Node.js 後重新開啟終端機 |
| `Permission denied` | Firestore 規則拒絕寫入或未啟用 Firestore | 檢查 `firestore.rules` 規則，或確認 Console 已完成資料庫初始化 |
| 執行指令權限錯誤 | Windows 執行原則限制 `.ps1` 指令檔執行 | 在 Windows PowerShell 中，一律使用 `npx.cmd` 代替 `npx` |
